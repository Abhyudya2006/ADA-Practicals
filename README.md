# 📘 ADA Practicals (Analysis and Design of Algorithms)

Welcome to **ADA-Practicals** — a curated collection of practical experiments and implementations for Algorithms & Data Structures (ADA), authored by **Abhyudya2006**.

This repository contains all core practicals performed during ADA coursework, including searching, sorting, greedy algorithms, graph algorithms, and dynamic programming techniques.

---

## 🚀 What This Repository Contains

Each folder corresponds to a separate practical, containing:

* 💻 **Source Code** — C/C++/Python implementations of algorithms  
* 🔍 **Step-by-Step Execution (NEW)** — Detailed dry-run output for better understanding  
* 📊 **Experiment Analysis** — data collection and results *(for selected practicals)*  
* 📈 **Graphs & Plots** — visual representation of time complexity *(where applicable)*  
* 📝 **Write-ups** — Aim, methodology, observations, and conclusions  

---

## ✨ New Enhancement

For the latest practicals, a different approach has been introduced:

> 🔎 Instead of focusing only on performance analysis, the code is designed to **display the complete execution (dry run)** of algorithms step-by-step.

### 💡 What this means:

* You can **see how the algorithm works internally**
* Helps in **better visualization and understanding**
* Useful for **learning, debugging, and presentations**
