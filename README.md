# 📘 ADA Practicals (Analysis and Design of Algorithms)

Welcome to **ADA-Practicals** — a curated collection of practical experiments and implementations for Algorithms & Data Structures (ADA), authored by **Abhyudya2006**.

This repository contains all core practicals performed during ADA coursework, including searching, sorting, greedy algorithms, and graph algorithms.

---

## 🚀 What This Repository Contains

Each folder corresponds to a separate practical, containing:

* 💻 **Source Code** — C++ implementations of algorithms
* 📊 **Experiment Analysis** — data collection and results
* 📈 **Graphs & Plots** — visual representation of time complexity
* 📝 **Write-ups** — Aim, methodology, observations, and conclusions

The practicals are organized chronologically and structured for clarity and learning.
